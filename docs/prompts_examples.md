
# Примеры промптов для Cursor 

Ниже приведены примеры промптов, которые использовались при разработке проекта. Каждый промпт направлен на решение конкретной небольшой задачи — написание функции, исправление бага, реализацию компонента сети, исследование гиперпараметров и т.п.

## Генерация синтетических данных (ADI метод)

1. `Добавь в ADI метод в файле @synthetic_data\src\adi.cpp источник тепла Q(x,y,z,t), моделирующий одноимпульсное лазерное облучение (гауссов луч по x,y и гауссов импульс по времени).`

2. `Проверь правильность реализации метода переменных направлений (ADI) для решения 3D уравнения теплопроводности с адиабатическими граничными условиями из файла @synthetic_data\src\adi.cpp. Область: параллелепипед 40×10×5 мм. Шаги по времени и пространству задаются параметрами.`

3. `Измени параметры сетки координат (x,y,z,t) из файла @synthetic_data\src\grid.cpp, чтобы выполнялось условие сходимоси метода (число Фурье меньше 0.5).`

4. `Вычисли температуру из ADI решения. Выходные данные нужно сохранить в формате CSV с колонками x,y,z,t,T.`

## Архитектура PINN и hard constraint

5. `Добавь в файле @pinn\physics\heat_source.py функцию источника тепла Q(x,y,z,t), моделирующую многоимпульсное лазерное облучение (гауссов луч по x,y и гауссов импульс по времени). Добавь возможность менять количество импульсов через @configs`

6. `Реализуй полносвязную сеть на PyTorch с 5 слоями по 100 нейронов, активация tanh. Входные признаки: x,y,z,t нормируй в (0,1)). Выход — температура. Используй библиотеку DeepXDE и Pytorch.`

7. `Как сделать hard constraint для начального условия T(x,0)=T0? Напиши в файл @pinn\models\constraints.py преобразование выхода сети: T_hat = T0 + t * y, где y — выход сети. Проверь, что при t=0 T_hat = T0.`

8. `Попробуй другое hard constraint: T_hat = T0 + (1 - exp(-t/tau)) * Softplus(y). Напиши код для этого преобразования. Пусть tau будет гиперпараметром.`

9. `Добавь Fourier features для пространственных и временной координаты (x,y,z, t) перед подачей в сеть. Используй 5 частот с random normal инициализацией. Сравни с обычным входом.`

## Функция потерь (Loss) для PINN

10. `Напиши функцию потерь для PINN: L = λ_pde * L_pde + λ_ic * L_ic + λ_bc * L_bc. L_pde — среднеквадратичная невязка уравнения теплопроводности (с использованием autograd для градиентов). L_ic и L_bc — для начальных и граничных условий. Используй библиотеку DeepXDE.`

11. `Как вычислить градиенты второго порядка (∂²T/∂x²) в PyTorch для функции потерь PDE? Покажи код с torch.autograd.grad или torch.func.hessian.`

## Многоимпульсный режим

12. `Реализуй источник тепла для PINN для многоимпульсного лазерного облучения в @pinn\physics\heat_source.py: последовательность гауссовых импульсов с периодом 20 с, FWHM длительностью 1 мс, амплитудой 5 Вт/мм³. Напиши функцию Q(x,y,z,t).`

13. `Как разделить временные и пространственные переменные в PINN для моделирования многоимпульсного нагрева? Предложи архитектуру: для пространства — сеть с Fourier features, для времени — отдельная сеть или RNN, затем объединение. Поищи подобные примеры в библиотеке DeepXDE.`

## Обратная задача

14. `Модифицируй PINN для решения обратной задачи: коэффициент теплопроводности k неизвестен. Сделай k обучаемым параметром через библиотеку DeppXDE. Используй синтетические данные температуры на глубинах 0, 1, 3 мм как наблюдения. Добавь loss для соответствия данным.`

15. `Как восстановить k с точностью до 1%, если ρ и cp известны? Напиши цикл обучения, где loss включает PDE residual и наблюдения реальных температур. После обучения выведи найденное значение k и график его изменения по эпохам.`

## Гиперпараметры и инициализация

16. `Сравни функции активации (tanh, sigmoid, ReLU, sin, swish) для PINN. Напиши скрипт, который обучает 5 моделей с одинаковыми остальными параметрами и выводит RMSE на тестовой выборке для каждого набора параметров.`

17. `Протестируй инициализацию Glorot normal vs Glorot uniform для нашей задачи. Какая даёт более быструю сходимость? Напиши код для эксперимента.`

18. `Подбери λ_pde, λ_ic, λ_bc так, чтобы вклады в loss были сбалансированы (например, через обучение с soft adapt). Реализуй автоматическое взвешивание компонентов loss.`

## Визуализация и метрики

19. `Построй 2D визуализацию температурного поля в плоскости xz параллелепипеда 40×10×5 мм в несколько моментов времени моделирования через @visualize_results.py. Используй matplotlib или plotly. Цветовая шкала — температура в градусах Цельсия.`

20. `Напиши функцию для вычисления RMSE между предсказанием PINN и ADI решением на тестовой сетке. Добавь вывод средней ошибки по области и по времени.`
